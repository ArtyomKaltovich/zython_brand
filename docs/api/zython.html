solve_satisfy(*, all_solutions=False, result_as=None, verbose=False)¶

Finds solution that satisfied constraints, or the error message if the model can’t be solved

Parameters

  • all_solutions (bool) – If True all solutions will be returned. If False only the first solution is returned. Default values is False, so the model will return only one solution, as finding all of them can be calculation hard

  • verbose (bool) – If True the source code of the model will be print to stdout

Returns

Result – result of the model solution, value of variables can be reached by dict syntax.
